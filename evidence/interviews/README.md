# Interview Write-ups

Each interview file should include:

1. **Who**: name/role of the person interviewed, their organisation
2. **When**: date of the interview
3. **Key quotes**: direct quotes from the interview
4. **What we learned**: summary of findings
5. **What changed in the app**: specific features added or modified as a result
