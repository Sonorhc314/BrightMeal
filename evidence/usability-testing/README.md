# Usability Testing

Each team member runs 1 test session with someone outside the team using the live app at [brightmeal.vercel.app](https://brightmeal.vercel.app). Copy `template.md` and fill it in.

**Naming:** `test-[your-name]-[date].md` (e.g. `test-rawand-2026-04-15.md`)

**Target:** 5 completed tests minimum.
