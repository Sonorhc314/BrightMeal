# Usability Test Report

## Details

- **Conducted by:** Rawand Ali
- **Date:** 10/04/2026
- **Participant:** Family member, hasn't used anything like this before
- **Role tested:** Donor
- **Device:** Phone
- **Duration:** ~10 minutes

## Tasks

1. Sign up and create a donor profile
2. Post a new donation with allergens and a pickup window
3. Check the leaderboard and badge progress
4. View donation history

## Observations

| # | Task | Completed? | Difficulty (1-5) | Notes |
|---|------|-----------|-----------------|-------|
| 1 | Create profile | Yes | 1 | Went through it fine |
| 2 | Post a donation | Yes | 2 | Got through it but got annoyed at the date picker when trying to set it for today |
| 3 | Check leaderboard | Yes | 1 | Made sense to him straight away |
| 4 | View history | Yes | 1 | No problems |

## Direct Quotes

- "It's quite inconvenient that there is no option for today because most donors will probably upload it for today so that's the main concern."

## Issues Found

1. **Medium** — No quick way to set dates for today. You have to scroll through the date picker even if the food is ready right now. Most donors would be posting same-day surplus so this slows them down for no reason.

## Suggestions from Participant

- Some kind of "Today" button for the date fields so you don't have to manually pick the date every time

## Reflections

He got through everything without needing help which is good. The only real complaint was the date picker thing, and honestly he's right — if you're a bakery posting leftover bread at 4pm you just want to hit "today" and set a time, not scroll through a calendar. We'd probably add quick-select buttons like "Today" and "Tomorrow" above the date fields if we had more time.
